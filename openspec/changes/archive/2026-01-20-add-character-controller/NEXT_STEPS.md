## 🎯 后续可选功能清单

### 1. 空中移动增强（优先级：高）

#### 1.1 二段跳
**描述**: 在空中可以再跳一次  
**工作量**: 1-2 小时  
**实现要点**:
- 添加 `MaxAirJumps` 配置（默认 1）
- 添加 `AirJumpHeightMultiplier` 配置（默认 0.8，第二次跳跃高度降低）
- 添加 `_airJumpCount` 计数器
- 地面时重置计数器
- 空中跳跃时检查计数器

**配置示例**:
```csharp
[Header("Air Jump Settings")]
public bool EnableAirJump = false;
public int MaxAirJumps = 1;
public float AirJumpHeightMultiplier = 0.8f;
```

#### 1.2 空中冲刺
**描述**: 在空中使用冲刺  
**工作量**: 30 分钟  
**实现要点**:
- 修改 `Dash()` 方法，移除地面检测限制
- 添加 `AllowAirDash` 配置选项
- 可选：空中冲刺消耗空中跳跃次数

#### 1.3 滑翔
**描述**: 长按跳跃键在空中滑翔，减缓下落  
**工作量**: 1 小时  
**实现要点**:
- 添加 `EnableGlide` 配置
- 添加 `GlideGravityMultiplier` 配置（如 0.3，重力降低到 30%）
- 检测跳跃键持续按下
- 修改空中重力应用
