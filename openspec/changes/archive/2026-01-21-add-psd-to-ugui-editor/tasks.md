# Implementation Tasks

## 1. 环境准备和依赖
- [ ] 1.1 调研并选择合适的PSD解析库（推荐：Ntreev.Library.Psd 或 PSD-Parser）

## 2. PSD数据模型和解析
- [ ] 2.1 创建 `PsdLayerInfo` 数据类，存储图层信息（名称、位置、大小、类型、可见性）
- [ ] 2.2 创建 `PsdTextLayerInfo` 数据类，存储文本图层特有信息（字体、大小、颜色、对齐、效果）
